# Cadmium::Util

A collection of useful utilities used internally in Cadmium.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     cadmium_util:
       github: cadmiumcr/utilities
   ```

2. Run `shards install`

## Usage

```crystal
require "cadmium_util"
```
